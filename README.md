# Kingdom Packer

Because the free tier of aws lets you have 1 T2.micro for free, without charge, we're creating a packer image that puts all my services on a single T2.micro box.

## Services

- Jenkins
- GitLab
- Mumble
- Nginx
- App: discourse, tournement, mission app
